# 🎮 KidLearn Play - Interactive Learning Game for Kids

An interactive educational game app for kids aged 2-7, built with Flutter, featuring:
- 📚 Number learning (1-20)
- 🎮 Mini games (Tap count, Drag match, Memory)
- 🎨 Animated storytelling
- 💰 Dynamic ad management via Admin Dashboard

## 📁 Project Structure

```
kidlearn-play/
├── flutter_app/              # Flutter mobile app
├── admin_dashboard/          # React admin panel
├── backend/                  # Node.js API (Firebase)
├── docs/                     # Documentation
└── README.md
```

## 🚀 Getting Started

### Flutter App
```bash
cd flutter_app
flutter pub get
flutter run
```

### Admin Dashboard
```bash
cd admin_dashboard
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm start
```

## 🏗️ Tech Stack

- **Mobile:** Flutter
- **Admin:** React.js
- **Backend:** Firebase + Cloud Functions
- **Database:** Firestore
- **Ads:** Adsterra
- **State Management:** Provider (Flutter)
